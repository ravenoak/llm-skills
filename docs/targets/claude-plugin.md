# Target: claude-plugin

The repo is a single Claude Code plugin called `llm-skills` that bundles every skill whose `targets.claude-plugin.enabled === true`. Skills are auto-discovered by Claude Code from `skills/<id>/SKILL.md`; there is no `skills` field in `plugin.json`.

Two generated files:

| Path | Source |
|---|---|
| `.claude-plugin/plugin.json` | repo metadata in `tools/ts/src/cli.ts#readRepoMeta` |
| `.claude-plugin/marketplace.json` | same metadata + one self-referencing plugin entry pointing at `./` |

`.claude-plugin/marketplace.json` is the marketplace index that Claude Code reads for `/plugin marketplace add`. It always lists exactly one plugin (`llm-skills`) with `source: "./"`. Both files are regenerated by `skillsmith build`; never hand-edit them.

Skill-bundled commands, agents, and hooks (placed under `skills/<id>/overrides/claude-plugin/{commands,agents,hooks}/`) are merged into root-level directories during build.
